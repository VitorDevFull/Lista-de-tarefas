# Lista de Tarefas

Projeto de uma lista de tarefas desenvolvido com HTML, CSS e JavaScript.

A aplicação permite adicionar, concluir, editar e excluir tarefas. Também possui contadores de tarefas totais e concluídas, além de salvar os dados no navegador usando localStorage.

## Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Desfazer conclusão de uma tarefa
- Editar tarefas
- Excluir tarefas
- Contador de tarefas totais
- Contador de tarefas concluídas
- Mensagem quando não há tarefas cadastradas
- Salvamento das tarefas no navegador com localStorage
- Layout responsivo para telas menores

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
